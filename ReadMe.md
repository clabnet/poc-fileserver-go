# UPLOAD with GoLang

### Setup environment

```
go get -u github.com/gorilla/mux
```

### Start development server

```
cd file-upload-go
go run main.go
```

### Open test page

```
http://localhost:3000/file-upload-fe/fileupload.html
```

> Please note : start a simple web server on port 3000
